Flutter handoff checklist
  • Primary CTA always visible on attendance critical screens.
  • Every error state has direct retry action.
  • Offline queue state visible when pending > 0.
  • Touch target minimum 48dp for accessibility.
  • Adaptive layout for large screens and foldables.
  • Components mapped to reusable Flutter widgets.
