Page 4 - Prototype and Handoff

Navigation prototype and Flutter handoff rules.

Login-> Dashboard-> Attendance-> History-> Recap-> Sync

Transitions: fast fade + slide. Keep interaction latency under 300ms where possible.

Flutter handoff notes
  • Spacing scale: 8/12/16/24/32
  • Corner radii: 10/14/20
  • Text tokens: display, heading, title, body, caption
  • Core components: button, text field, card, tag, async states
  • Accessibility: contrast AA, minimum touch target 44x44
  • Offline-first behavior: queue and retry sync visible in UI
